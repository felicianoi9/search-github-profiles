# Projeto de estudo - Dio - Busca de perfil Github

Este projeto tem por objetivo trabalhar conceitos e melhores práticas de compomentização. O resultado final é uma página simples de consulta de perfil pelo nome de usuário do Github. Ao realizar a pesquisa é exibido informações como: nome, username, foto e lista de seus projetos públicos.

## O que foi realizado a mais:

Os links para os repositórios foram corretamente passados para o componente.

## Como usar:
### Clone o projeto:
> git clone https://github.com/felicianoi9/search-github-profiles.git
### instale as dependêmcias:
> yarn install
### Enjoy!!!
> yarn start
